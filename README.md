# 🎓 EduNext - Modern E-Learning Platform

EduNext adalah platform manajemen kursus online sederhana yang dibangun menggunakan **CodeIgniter 4**, **Docker**, dan **Bootstrap 5**. Aplikasi ini memiliki UI dashboard dengan tema **Premium Dark & Modern Blue** untuk memberikan pengalaman pengguna yang elegan dan profesional.

## ✨ Fitur Utama
- **Role-Based Access Control**: Login khusus untuk Admin dan User.
- **Admin Dashboard**: Fitur manajemen materi (Upload & Hapus).
- **User Dashboard**: Daftar materi kursus yang tersedia.
- **Modern UI**: Desain minimalis dengan efek Glassmorphism dan skema warna Dark Mode.
- **File Support**: Mendukung materi berupa Video dan Gambar/Foto.

## 🚀 Teknologi yang Digunakan
- **Backend**: PHP 8.1+ (CodeIgniter 4)
- **Frontend**: Bootstrap 5, Font Awesome 6, Google Fonts (Sora & DM Sans)
- **Database**: MySQL / MariaDB
- **Infrastructure**: Docker & Docker Compose
