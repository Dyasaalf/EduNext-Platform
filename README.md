# 🎓 EduNext - Modern E-Learning Platform

EduNext adalah platform manajemen materi kursus online yang dirancang dengan antarmuka pengguna (UI) modern dan minimalis. Proyek ini dibangun menggunakan **CodeIgniter 4** sebagai backend dan sistem **Custom CSS Modern** untuk memberikan pengalaman visual yang eksklusif dan performa yang ringan.

## ✨ Fitur Utama
- **Role-Based Access Control (RBAC)**: Sistem masuk terpisah untuk Admin dan Pengguna (Siswa).
- **Dashboard Admin**: Panel khusus untuk mengunggah, mengelola, dan menghapus materi kursus.
- **Daftar Materi Dinamis**: Pengguna dapat melihat daftar materi terbaru dengan label kategori (Video/Gambar).
- **Modern UI/UX**: 
    - **Dark Mode Concept**: Skema warna gelap yang nyaman di mata.
    - **Glassmorphism Design**: Efek kartu transparan dengan blur di latar belakang.
    - **Minimalist Sidebar**: Navigasi fokus hanya pada fungsi utama.
- **Manajemen File**: Sistem upload file yang terintegrasi dengan penyimpanan lokal.

## 🚀 Teknologi yang Digunakan
### Backend
- **PHP 8.1+**
- **CodeIgniter 4** (Framework)

### Frontend (No Framework)
- **Custom CSS3**: Menggunakan teknik *CSS Grid*, *Flexbox*, dan *Variables* tanpa framework Bootstrap agar lebih ringan dan unik.
- **Font Awesome 6**: Untuk kebutuhan ikonografi.
- **Google Fonts**: 
    - *Sora*: Untuk tipografi judul yang tegas.
    - *DM Sans*: Untuk keterbacaan teks utama yang optimal.

### Infrastructure
- **Docker & Docker Compose**: Untuk standarisasi lingkungan pengembangan.
- **MySQL**: Sebagai sistem manajemen basis data.
